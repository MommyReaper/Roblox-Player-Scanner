# Advanced Roblox Player Scanner 🎮

## Features
- 🖱️ Draggable GUI interface
- 📊 Real-time player information gathering
- 📍 Position tracking and health monitoring
- 🏃 Teleport and follow functionality
- 💎 VIP/Premium member detection
- 📋 Data export capabilities
- 🔄 Auto-refresh every 3 seconds

## Information Gathered
- Player usernames, display names, and User IDs
- Account age and membership type
- Real-time position and health status
- Team information and character stats
- Inventory items and leaderstats
- Body parts and accessories

## Usage
1. Execute script in Roblox using a compatible executor
2. Drag the GUI by clicking the title bar
3. Use buttons to interact with player data
4. Auto-refresh keeps information current

## Supported Executors
- Synapse X
- KRNL
- Fluxus
- other popular Roblox executors
